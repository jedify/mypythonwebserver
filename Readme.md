# Building REST apis with Nodejs without using any framework or external library

## Start the server
$ python server.py

Usage of LIST: `$curl -X GET 127.0.0.1:8080`

## For Instance,
	$curl -X GET 127.0.0.1:8080/api/message
```json
{ "message": "Hello World" }
```

# Access UI
http://localhost:8080/index.html
